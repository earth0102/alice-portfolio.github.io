Entity Relationship Diagram (ERD)

For a Data Analyst Bootcamp project, we needed an effective way to input machine statuses, the material used or any accident reported to a shift report. We suggested using online/web forms to fill in necessary information instead of hand-writing and Excel input. The idea was not only to increase the data accuracy but also to make the data consistent and make it more readable and visualised. In this case, I acted as a Database Administrator to analyse the fields in a shift report; create an ERD; prepare SQL queries; and sample data for dashboard visualisation.
